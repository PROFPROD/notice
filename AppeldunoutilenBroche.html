Appel d'un outil en Broche

Les avances doivent être ouvertes


Étape 1

Choisir le mode "MDI"


Méthode 1 (code ISO)

Cette méthode est utilisée pour appeler n'importe quel emplacement. Ce code est utilisé dans les programmes.


Étape 1

Taper le code ISO "T(numéro outil) M6" et valider par "ENTER"


Exemple pour mettre en broche l'outil placé dans la poche 4:


T4 M6 


Étape 2

Appuyer sur "DEPART CYCLE" pour lancer le processus

Cette méthode est utilisée pour appeler l'outil suivant ou précédent.

Méthode 3 (Appel outil le + rapide)

Cette méthode est utilisée pour appeler n'importe quel emplacement rapidement


Étape 1
